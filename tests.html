This document is used to test the behavior of Chunks. As it is (unfortunately) not feasible to automatically test the rules, tests must be validated manually.

For each test:

  1. Read the description and text contained in the test.
  2. Observe the test appearance.
  3. If the test is displayed in accordance with the description, the test passes. Otherwise, it fails.

Ideally, tests should be run in every browser.

Each individual test is contained in a box with red dotted borders like this one.
